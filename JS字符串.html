字符串和字符串的操作

每种语言都会有一个hello world 
var hellowworld="hello world";
alert(hellowworld)

字符串

字符串的连接 可以用 + 连接 
var name = '小明';
var age = 20;
var message = '你好，' + name + '，你今年' + age + '岁了';
alert(message);
当然还可以这样 
var message1=`你好, ${name}, 你今年${age}岁了!`;
alert(message1);
注意第一行，不是''，而是`` 为什么从message变成message1了呢，因为字符串初始化后值不会变，再次赋值不会报错，但是也会无效 字符串的操作 要获取字符串某个指定位置的字符，使用类似数组的下标操作 
alert(helloworld[0])
alert(helloworld[0]) toUpperCase()能把字符串全部变成大写 
alert(helloworld.toUpperCase())
注意string.UpperCase是个函数，尾巴上有个括号
同理toLowerCase()能把字符串全部变成小写 
alert(helloworld.toLowerCase())
indexOf()会搜索指定字符串出现的位置 
alert(helloworld.indexOf('o'))
注意是单引号''不是双引号""，是因为<pre onclick="alert(helloworld.indexOf('o'))">里onclick就有双引号，.indexOf()的参数还用双引号的话，浏览器不认啊
如果指定字符没有，那么值就是-1
substring()返回指定索引区间的子串 
alert(helloworld.substring(2,7))
两个参数是2,7，返回值是helloworld[2]到helloworld[6]形成的子串
如果两个参数是7,2，返回值还是一样的
alert(helloworld.substring(7))
只有一个参数7，返回值是helloworld[7]到helloworld这个字符串结束形成的子串
